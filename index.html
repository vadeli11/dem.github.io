Доменное имя моего сайта — vadeli11.github.io — состоит из следующих уровней иерархии:

  1. Домен верхнего уровня (TLD, уровень 1): .io
    Это международный домен верхнего уровня.
  2. Домен второго уровня (уровень 2): github.io
    Принадлежит GitHub и используется для хостинга пользовательских сайтов.
  3. Поддомен третьего уровня (уровень 3): vadeli11.github.io
    Здесь vadeli11 — это имя моего аккаунта на GitHub.

Таким образом, полное доменное имя строится по принципу:
[имя_пользователя].[платформа].[TLD].
